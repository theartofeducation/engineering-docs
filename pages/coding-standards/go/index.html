    Go

    Built-In Tools#

    Go has tools that are built-in to the platform to handle things like code formatting structure and static analysis for code best practices. The following built-in tools are used for this:

    • goimports
    • gofmt
    • golint
    • govet

    Effective Go#

    We also use Effective Go as a general guide for best practices in our Go code. We strongly encourage our developers to get familiar with the concepts promoted in this document, and put them to practice in their code.
